# 목동유쌤영어학원 2026 중등부 여름 무료특강 통합 신청 웹앱

초6~중3 대상 7개 무료특강을 한 페이지에서 비교하고, 필요한 수업을 여러 개 선택해 **한 번의 신청서로 Google Sheets에 접수**하는 모바일 중심 웹앱입니다.

- 프런트엔드: HTML · CSS · JavaScript 정적 웹페이지
- 배포: Vercel
- 신청 저장: Google Apps Script Web App
- 데이터 관리: Google Sheets
- 별도 유료 서버·DB: 필요 없음
- 기존 개별 Google Form: 데이터 참고용으로 코드에 보관되며 학부모 화면에는 노출하지 않음

## 2. 완성된 기능

### 학부모·학생 화면

- 실제 첨부 포스터 7종 적용
- 포스터 확대 보기
- 전체 / 선택한 특강 / 신청 가능 필터
- 여러 특강 동시 선택
- 김지연T 참여 시간 선택
- 학생 정보·보호자 정보 한 번만 입력
- 재원생 / 외부생 구분
- 개인정보 수집·이용 동의
- 최종 확인 화면
- 접수번호 자동 생성
- 중복 제출 방지용 요청 ID
- 선택 목록 24시간 임시 저장
- 모바일·태블릿·PC 반응형

### Google Sheets 관리자 화면

`setupSpreadsheet()`를 처음 한 번 실행하면 아래 시트가 자동 생성됩니다.

1. **신청내역**
   - 접수번호, 접수시각, 학생·보호자 정보, 선택 특강, 일정·선택 시간, 상담 메모 저장
   - 처리상태 드롭다운: 신규접수 / 상담중 / 상담완료 / 등록완료 / 대기 / 취소
   - 동일 요청 ID 중복 저장 차단

2. **특강별명단**
   - 한 번의 통합 신청을 선택 특강별로 한 행씩 자동 분리
   - 선생님별·특강별 필터링과 출석 명단 정리에 바로 활용
   - 특강별 처리상태 드롭다운 제공

3. **특강설정**
   - 교사명, 특강명, 대상, 일정, 수강료, 정원, 상태, 포스터, 소개문구 수정
   - `상태`를 `마감`으로 바꾸면 웹 선택 버튼 자동 비활성화
   - `정원`에 숫자를 입력하면 접수 시 정원 초과 방지
   - `세부선택지(|구분)`에 `오후 2시|오후 8시`처럼 입력하면 신청서에 선택 메뉴 생성

4. **환경설정**
   - 캠페인명, 대표전화, 신규 신청 알림 이메일 관리

# 4. Google Sheets 연동

웹 코드와 Apps Script 코드는 완성되어 있습니다. 다만 Google 계정 내부에 실제 Sheet를 만들고 권한을 승인하는 단계는 계정 소유자가 한 번 진행해야 합니다.

## 4-1. 새 Google Sheet 만들기

1. Google Drive에서 **새로 만들기 → Google 스프레드시트**를 누릅니다.
2. 파일 이름을 `2026 중등부 여름 무료특강 통합 신청내역` 등으로 지정합니다.
3. 상단 메뉴에서 **확장 프로그램 → Apps Script**를 누릅니다.

## 4-2. Apps Script 붙여넣기

1. Apps Script 편집기의 `Code.gs` 기존 내용을 모두 지웁니다.
2. 이 프로젝트의 `google-apps-script/Code.gs` 내용을 전부 복사해 붙여넣습니다.
3. 저장합니다.

`appsscript.json`은 선택 사항입니다. `Code.gs`만으로도 동작합니다.

## 4-3. 시트 자동 생성

1. Apps Script 상단 함수 선택 메뉴에서 `setupSpreadsheet`를 선택합니다.
2. **실행**을 누릅니다.
3. 처음 실행 시 Google 권한을 승인합니다.
4. Sheet로 돌아가 아래 4개 탭이 생성됐는지 확인합니다.
   - 신청내역
   - 특강별명단
   - 특강설정
   - 환경설정

특강설정 값을 처음 상태로 다시 돌려야 할 때만 `resetCourseSettingsToDefaults`를 실행합니다. 이 함수는 현재 특강설정 값을 첨부 포스터 기준 기본값으로 덮어씁니다.

## 4-4. 웹 앱 배포

1. Apps Script 우측 상단 **배포 → 새 배포**
2. 유형 선택에서 **웹 앱**
3. 설정
   - 설명: `2026 중등부 여름 무료특강 통합 신청서`
   - 다음 사용자로 실행: **나**
   - 액세스 권한: **모든 사용자**
4. **배포**
5. `https://script.google.com/macros/s/.../exec` 형태의 웹 앱 URL 복사

Apps Script 코드를 나중에 수정했다면 **배포 → 배포 관리 → 수정 → 새 버전 → 배포**를 해야 실제 웹 앱에 반영됩니다.

## 4-5. 웹페이지와 연결

`config.js`에서 아래 값만 교체합니다.

```js
googleScriptUrl: 'PASTE_GOOGLE_APPS_SCRIPT_WEB_APP_URL_HERE',
```

예시:

```js
googleScriptUrl: 'https://script.google.com/macros/s/AKfycbxxxxxxxx/exec',
```

이후 Vercel에 다시 배포하면 신청서가 Google Sheets에 저장됩니다.

# 5. Vercel 배포

## GitHub 연동 방식

1. GitHub 새 저장소를 만듭니다.
2. 이 프로젝트 폴더 내부 파일을 저장소 최상단에 업로드합니다.
3. Vercel에서 **Add New → Project**
4. 저장소 선택
5. Framework Preset: **Other**
6. Build Command·Output Directory: 비워 둠
7. **Deploy**

## Vercel CLI 방식

Windows 명령 프롬프트에서 프로젝트 폴더로 이동한 뒤:

```bash
npx.cmd vercel
```

운영 주소로 배포:

```bash
npx.cmd vercel --prod
```

# 6. 배포 후 테스트

1. 휴대전화에서 Vercel 주소를 엽니다.
2. 특강 2개 이상을 선택합니다.
3. 김지연T를 선택했다면 참여 시간을 지정합니다.
4. 학생·보호자 정보를 테스트 값으로 입력합니다.
5. 개인정보 동의와 최종 확인에 체크합니다.
6. 신청서를 제출합니다.
7. 완료 화면에서 접수번호를 확인합니다.
8. Google Sheet `신청내역`에 한 행이 생기고, `특강별명단`에는 선택한 특강 수만큼 행이 생겼는지 확인합니다.
9. 같은 제출이 중복 행으로 저장되지 않았는지 확인합니다.
10. 테스트 행을 삭제합니다.
11. `특강설정`의 한 특강 상태를 `마감`으로 바꾼 뒤 웹에서 선택이 차단되는지 확인합니다.
